# 🥥 Coconut Supply Chain & Financial Ledger System

A Full-Stack web application and Data Analytics dashboard designed to track logistics, calculate real-time profit margins, and manage cloud-based accounting records for a coconut distribution business.

## 🚀 Features
- **Mobile-First Data Entry:** Responsive UI built with Bootstrap 5 for easy on-site recording via smartphones.
- **Automated Profit Calculation:** Built-in business logic to calculate total revenue, transportation costs, and net profit instantly.
- **Cloud Database Integration:** Uses Google Apps Script (REST API) to send data directly to Google Sheets without manual data entry.
- **Real-Time BI Dashboard:** Connected to Looker Studio for data visualization and financial reporting.
- **Export to CSV:** Allows users to download raw data for further pivot table analysis.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5, Vanilla JavaScript
- **Backend & API:** Google Apps Script
- **Database:** Google Sheets
- **Data Analytics:** Looker Studio
- **Deployment:** GitHub Pages

## 💡 The Motivation
Coming from an accounting background, I realized the pain point of manual data entry and delayed month-end reporting. I built this system to automate the workflow, bridge the gap between field operations (4-wheel truck logistics) and the accounting desk, and provide real-time financial insights.
